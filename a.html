 As cores são introduzidas através do elemento usando
o sistema RGB, assim, um trecho de texto pode ter
uma cor diferente da definição geral de cores, feita
através dos atributos.		 A formatação permite que o autor do documento altere
o tamanho das letras em trechos específicos do texto.
Para que possamos mudar de linha, precisamos usar
comandos em HTML para que o browser realmente
mude de linha (ele não respeita mudanças de linha visuais do texto escrito em HTML).
